# Campus-Navigation-System-C-2.0
大学校园导游系统（C语言 · 数据结构课程设计）

本项目使用 C 语言 + 邻接矩阵图结构 实现东莞理工学院（国际创新区 + 本部）校园导航系统，满足课程要求：

景点（≥20个）管理：增删改查

道路（边）管理：增删改查

最短路径（Dijkstra）

所有路径（DFS 枚举）

多景点最佳游览路径（TSP 近似：最近邻 + 2-opt）

文件输入输出（TXT & CSV）

区分道路类型（机动车 / 人行道）

菜单界面

📌 功能特性

图结构：无向带权图（邻接矩阵）

节点信息：名称 / 简介

路径权重：距离（你可根据测距补全）

扩展功能：

CSV 导入导出

路径类型过滤

近似最优游览路径
